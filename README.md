## ğŸš€ Quick Start

### Prerequisites
```bash
node >= 14.x
npm >= 6.x
```

### Installation

1. **Clone & Install**
```bash
git clone https://github.com/Lavin-kulal/whatsapp-mern.git
cd whatsapp-mern
```

2. **Backend Setup**
```bash
cd whatsapp-backend
npm install
```

Create `.env`:
```env
PORT=9000
connection_url=your_mongodb_uri
PUSHER_APP_ID=your_app_id
PUSHER_KEY=your_key
PUSHER_SECRET=your_secret
PUSHER_CLUSTER=ap2
```

```bash
npm start  # Runs on :9000
```

3. **Frontend Setup**
```bash
cd ../whatsapp-frontend
npm install
npm start  # Runs on :3000
```
